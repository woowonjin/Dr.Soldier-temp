그런데 군인을 위한 앱은 도대체 어디있는걸까요?

핵심 기술 소개

최선의 기술을 모아서 최고의 가치를 만들고 최상의 경험을 선사합니다.

  • 카카오/애플 로그인

    사용자를 위해 회원가입이 필요없이, 보다 안전하고 편한 로그인 서비스를 제공합니다.

  • Django 서버

    컴퓨터 공학도라면 서버는 직접 구축해야죠! 직접 구축한 서버는 저희의 자랑입니다.

  • AWS 배포

    Django 서버를 AWS 서버에 배포하여 신뢰할 수 있는 서버, 저희가 직접 관리할 수 있는 서버환경을 만들었습니다.

  • Alamofire

    구축한 서버와의 HTTP 통신을 합니다.

  • SQLite3

    사용자의 개인정보는 로컬스토리지에만 저장됩니다. 외부와의 데이터교환 없이 보다 강력한 보안을 제공합니다.
